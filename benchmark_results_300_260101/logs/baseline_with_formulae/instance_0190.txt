To calculate the corrected QT interval (QTc) using the Bazett formula, we use the following formula:

**QTc Bazett: QTc = QT / sqrt(RR)**

Where:
- **QT** is the QT interval in milliseconds (msec)  
- **RR** is the RR interval in seconds (calculated as 60 / heart rate in bpm)  

### Given:
- Heart rate (HR) = 48 bpm  
- QT interval = 201 msec  

### Steps:
1. Calculate the RR interval:  
   RR = 60 / HR = 60 / 48 = 1.25 seconds  

2. Take the square root of RR:  
   sqrt(RR) = sqrt(1.25) ≈ 1.118  

3. Divide QT by sqrt(RR):  
   QTc = 201 / 1.118 ≈ 179.8
